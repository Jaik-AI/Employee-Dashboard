# Employee Dashboard

A Python project built using **Pandas** and **Matplotlib** to analyze employee data and create an HR analytics dashboard.

The project reads employee data from a CSV file, performs statistical analysis, and generates visualizations to help understand salary distribution, department statistics, and workforce insights.

---

## Features

- Highest Paid Employee
- Lowest Paid Employee
- Average Salary by Department
- Employees in Each Department
- Highest Salary
- Lowest Salary
- Average Age
- Most Experienced Employee
- Youngest Employee
- Oldest Employee
- Employees Above Average Salary
- Employees with More Than 5 Years Experience
- Top 3 Highest Paid Employees
- Department with Highest Average Salary

---

## Dashboard Visualizations

The dashboard includes six visualizations:

- 📊 Average Salary by Department
- 🥧 Employees in Each Department
- 🔵 Experience vs Salary
- 📊 Salary Distribution (Histogram)
- 📊 Department Salary Comparison
- 📊 Average Experience by Department

## How to Run

1. Install the required libraries

```bash
pip install pandas matplotlib
```

2. Place `employees.csv` inside the project folder.

3. Run

```bash
python main.py
```
